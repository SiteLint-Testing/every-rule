C7: Using CSS to hide a portion of the link text

Technique: C7. This technique supplements brief visible link text with additional text that is visually hidden but remains available to assistive technologies.

Related WCAG Success Criteria: 2.4.4 Link Purpose (In Context) and 2.4.9 Link Purpose (Link Only).

View the W3C C7 technique.

Passing example

This succeeds because the visible link text says Full story, while the visually hidden text adds the missing context. The combined accessible link text identifies the link's purpose without changing the visual design.

Washington has announced plans to stimulate economic growth. Washington stimulates economic growth Full story

Failing example 1: Generic link text with no supplemental hidden text

This fails to demonstrate C7 because the visible link text is only Full story, and no visually hidden text is included inside the link to make its purpose clear from the link text alone.

Philadelphia has announced plans to expand protected bike lanes. Full story

This text provides a valid same-page destination for the first failing example link.

Failing example 2: Supplemental text is hidden from assistive technologies

This fails because the supplemental text is hidden with display: none. That removes it from assistive technologies as well as from visual display, so the accessible link text remains only Full story.

New library hours will begin next month. New library hours begin next month Full story

This text provides a valid same-page destination for the second failing example link.

Failing example 3: Repeated generic link text points to different destinations

This fails to use C7 because each link is displayed only as Full story, with no visually hidden text inside the link to distinguish one destination from another. This is the type of pattern C7 is intended to improve.

City council approves a new tree-planting initiative. Full story

The school district releases its updated graduation plan. Full story

This text provides a valid same-page destination for the tree-planting story link.

Failing example 4: Repeated format links are not supplemented with the resource title

This fails to use C7 because multiple links are labeled only PDF, even though they point to different resources. A C7 implementation could visually hide each document title inside the corresponding link.

Accessibility checklist
PDF
Remediation guide
PDF

Failing example 5: Supplemental context is visually hidden, but placed outside the link

This fails to use C7 because the visually hidden supplemental text is present in the paragraph, but it is not included inside the anchor element. As a result, it does not become part of the link's accessible name.

Neighborhood park renovation Full story
