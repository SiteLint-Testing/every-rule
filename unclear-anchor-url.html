Rule ID: unclear-anchor-url

    Rule title: Links must have a clear, valid destination.

    Documentation: SiteLint documentation

    Code validator: View source on Bitbucket

    Jira Ticket: PM-1101

    Rule Test: unclear-anchor-url

    Here is an example of an unclear anchor URL that breaks accessibility rules:

    Link with href="#"

    Learn more about our accessibility tools

    Link with href="javascript:;"

    Read about JavaScript-based navigation

    Link with empty href

    Return to the top of the page

    These links should be avoided as they may be misinterpreted by screen readers or the action may be unexpected.
