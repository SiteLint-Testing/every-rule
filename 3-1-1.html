WCAG 3.1.1: Language of Page

View main list

Brief summary: The default human language of each web page must be programmatically determinable.

View the W3C Understanding page for WCAG 3.1.1 Language of Page.

Reference passing patterns

Passing pattern 1: English page language

A page written primarily in English can declare English on the root html element. 

<html lang="en">

Passing pattern 2: French page language

A page written primarily in French can declare French on the root html element. 

<html lang="fr">

Manual review patterns

Manual pattern 1: Language attribute exists but may be wrong

A page may include a language declaration while still choosing the wrong language for the main content. 

<html lang="en">

Bienvenido al centro de ayuda. Aqui puede revisar su cuenta, facturas y solicitudes de soporte.

Manual pattern 2: Language tag may be invalid

A page may include a language value that is a plain-language name rather than a valid language tag. 

<html lang="english">

Manual pattern 3: Parts of the page use another language

Language changes inside the page are related, but they are evaluated under WCAG 3.1.2 Language of Parts rather than the page-level language requirement.
