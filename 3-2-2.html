WCAG 3.2.2: On Input

View main list

Brief summary: Changing the setting of a user interface component must not automatically cause a change of context unless the user has been advised before using the component.

View the W3C Understanding page for WCAG 3.2.2 On Input.

Observed SiteLint behavior

SiteLint uses on-input as a manual-verification trigger for WCAG 3.2.2 and reports missing-submit-button for the form that cannot be submitted with an explicit button. It does not appear to determine whether input changes actually trigger unexpected navigation, submission, or other context changes.

The form controls on this page include autocomplete attributes where possible so unrelated WCAG 1.3.5 and best-practice autocomplete findings do not obscure 3.2.2 behavior.

Recommendation

Keep missing-submit-button as a concrete partial check for forms that rely on pointer or implicit submission. Treat on-input as a manual prompt unless SiteLint can inspect or execute change/input handlers and distinguish permitted updates from changes of context.

Passing examples

Passing example 1: Selection waits for explicit submit

Changing the select value does not navigate or submit. The user activates a button to apply the change.

Passing example 2: User is advised before automatic update

The instruction tells the user that changing the value updates results automatically.

Changing the filter updates the result list automatically.

Failing examples

Failing example 1: Selection automatically navigates without warning

Changing the selected value causes navigation without prior instruction.

Failing example 2: Form has no submit button

The form relies on implicit or scripted submission rather than providing a visible submit button.

Failing example 3: Checkbox submits on change without warning

Changing the checkbox immediately submits the form.

Manual review examples

Manual example 1: Partial page updates

Some automatic changes may update content in place without changing context. Testing should distinguish ordinary content updates from context changes.
