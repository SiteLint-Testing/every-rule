Incorrect Technique for Hiding Content

This page demonstrates an incorrect technique for hiding content by using text-indent.

This content is hidden using text-indent: -10000px. Screen readers will follow this text out of the viewport, making it difficult for visually impaired users to navigate.

Description: This rule determines if there is a defined style text-indent with a value that positions the element out of the viewport.

Context: The technique text-indent: -10000px is still being used to hide content visually. However, the downside is that screen readers follow this, causing the focus to move out of the visible area.

How to fix it: Use a technique that hides the content visually but does not move the element outside the visible viewport. For example, you can use a Bootstrap class like .sr-only which ensures the content is hidden visually but remains accessible to screen readers.
